*A monthly roundup of news about Artificial Intelligence, Machine Learning and Data Science. This is an eclectic collection of interesting blog posts, software announcements and data applications I've noted over the past month or so.*

## Open Source AI, ML & Data Science News

[A review of the current state of the Julia project](https://www.infoworld.com/article/3284380/data-science/what-is-julia-a-fresh-approach-to-numerical-computing.html), including performance comparisons with Go, Python and R. 

[Python 3.7.0 has been released](https://pythoninsider.blogspot.com/2018/06/python-3.html). Also, [Guido van Rossum steps down](https://www.economist.com/science-and-technology/2018/07/21/python-has-brought-computer-programming-to-a-vast-new-audience) as the project's "Benevolent Dictator for Life".

[Tensorflow 1.9 released](https://github.com/tensorflow/tensorflow/releases/tag/v1.9.0), featuring a new getting started guide for Keras.

[Apache OpenNLP 1.9.0](https://opennlp.apache.org/news/release-190.html) released, adding several improvements to the natural language processing library.

[The PYPL Popularity of Languages Index](http://blog.revolutionanalytics.com/2018/06/pypl-programming-language-trends.html) ranks Python as the 1st and R as the 7th most popular programming languages.

[R 3.5.1 has been released](https://stat.ethz.ch/pipermail/r-announce/2018/000630.html).

## Industry News

[Google introduces Cirq](https://ai.googleblog.com/2018/07/announcing-cirq-open-source-framework.html), an open-source framework to simulate noisy intermediate-scale quantum computers.

[Hortonworks announces new partnerships](http://www.dbta.com/Editorial/News-Flashes/Hortonworks-Expands-Partnerships-with-Microsoft-Google-Cloud-and-IBM-125703.aspx) with Google, Microsoft and IBM to bring its big data platform to the cloud.

[IBM plans to release an annotated dataset of 1 million images](https://www.ibm.com/blogs/research/2018/06/ai-facial-analytics), with the goal of reducing bias in facial recognition systems.

[Amazon Sagemaker enhances](https://aws.amazon.com/about-aws/whats-new/2018/07/amazon-sagemaker-announces-enhancements-for-built-in-algorithms-and-frameworks?utm_source=riq&utm_campaign=3253&utm_term=the%20riq%20newsdesk) the DeepAR, BlazingText, and Linear Learner algorithms built into the machine learning framework.

A [series on Data Ethics](https://www.oreilly.com/tags/data-ethics) by Mike Loukides, Hilary Mason and DJ Patil. The most recent post proposes a [checklist for ethical data science practices](https://www.oreilly.com/ideas/of-oaths-and-checklists).

[Apple has combined its various artificial intelligence divisions](https://www.theverge.com/2018/7/10/17555652/apple-siri-ai-john-giannandrea-machine-learning-core-ml-teams) into a new structure led by recent hire John Giannandrea.

### Microsoft News

[Open data sets curated by Microsoft Research](https://www.microsoft.com/en-us/research/blog/announcing-microsoft-research-open-data-datasets-by-microsoft-research-now-available-in-the-cloud?WT.mc_id=DSAIroundup-newsletter-davidsmi) for computer vision, natural language processing and scientific analysis. 

[Microsoft acquires Bonsai](https://blogs.microsoft.com/blog/2018/06/20/microsoft-to-acquire-bonsai-in-move-to-build-brains-for-autonomous-systems?WT.mc_id=DSAIroundup-newsletter-davidsmi), a startup focused on using ML to train autonomous machines using simulated environments.

Microsoft President Brad Smith calls for public regulation and corporate responsibility in the [use of facial recognition technology](https://blogs.microsoft.com/on-the-issues/2018/07/13/facial-recognition-technology-the-need-for-public-regulation-and-corporate-responsibility?WT.mc_id=DSAIroundup-newsletter-davidsmi).

[ML.NET 0.3 released](https://blogs.msdn.microsoft.com/dotnet/2018/07/09/announcing-ml-net-0-3?WT.mc_id=DSAIroundup-newsletter-davidsmi) adding ONNX support, LightGBM learners, and more to the .NET machine learning library.

[Microsoft Speech Service SDK version 0.5.0](https://azure.microsoft.com/en-us/blog/speech-services-july-2018-update?WT.mc_id=DSAIroundup-newsletter-davidsmi) has been released, with support for long-running audio and automatic reconnections.

[Azure HDInsight now supports Apache Spark 2.3](https://azure.microsoft.com/en-us/blog/azure-hdinsight-now-supports-apache-spark-2-3?WT.mc_id=DSAIroundup-newsletter-davidsmi), improving performance for Python-based interactions.

[Azure Databricks now includes RStudio integration](https://azure.microsoft.com/en-us/blog/azure-databricks-provides-the-best-apache-spark-based-analytics-solution-for-data-scientists-and-engineers?WT.mc_id=DSAIroundup-newsletter-davidsmi) for R developers. The integration is described in detail at the [Databricks blog](https://databricks.com/blog/2018/06/27/rstudio-integration.html).

[The Microsoft Face API has significantly improved](https://blogs.microsoft.com/ai/gender-skin-tone-facial-recognition-improvement?WT.mc_id=DSAIroundup-newsletter-davidsmi) its ability to recognize gender across skin tones.

## Learning resources

Andrey Kurenkov's essay on the [limitations of reinforcement learning](https://thegradient.pub/why-rl-is-flawed). A [February 2018 blog post by Alex Irpan](https://www.alexirpan.com/2018/02/14/rl-hard.html) also provides some cautionary examples.

[A Developer's Guide to Building AI Applications](https://info.microsoft.com/ww-landing-ai-developers-bot-ebook.html??WT.mc_id=DSAIroundup-newsletter-davidsmi), a free e-book that provides a gentle introduction to the Microsoft AI Platform.

A tutorial on [adding machine learning to mixed reality applications](https://docs.microsoft.com/en-us/windows/mixed-reality/mr-azure-307??WT.mc_id=DSAIroundup-newsletter-davidsmi), using Azure ML Studio.

[How to read and write Microsoft SQL Server data in Python](https://tomaztsql.wordpress.com/2018/07/15/using-python-pandas-dataframe-to-read-and-insert-data-to-microsoft-sql-server) using Pandas dataframes.

[Object dectection via distributed deep learning on multiple GPUs](https://blogs.technet.microsoft.com/machinelearning/2018/06/20/how-to-do-distributed-deep-learning-for-object-detection-using-horovod-on-azure?WT.mc_id=DSAIroundup-newsletter-davidsmi), using Horovid on Azure Batch AI.

[How Artificial Intelligence Could Prevent Natural Disasters](https://www.wired.com/story/how-artificial-intelligence-could-prevent-natural-disasters), from Wired.

[A neural network for colorizing video](https://ai.googleblog.com/2018/06/self-supervised-tracking-via-video.html) also finds application in object tracking.

Computer vision with an embedded learning library from Microsoft Research used to [monitor endangered Purple Martin nests](https://news.microsoft.com/transform/disneys-smart-birdhouses-reveal-the-secret-lives-of-purple-martins?WT.mc_id=DSAIroundup-newsletter-davidsmi) at Disney Animal World.

[A diabetic retinopathy prediction application](https://blogs.technet.microsoft.com/machinelearning/2018/06/25/building-a-diabetic-retinopathy-prediction-application-using-azure-machine-learning?WT.mc_id=DSAIroundup-newsletter-davidsmi), based on image analysis with convolutional neural networks and implemented with Azure Machine Learning.

[Fashion similarity matching](https://blogs.technet.microsoft.com/machinelearning/2018/07/10/how-to-use-siamese-network-and-pre-trained-cnns-for-fashion-similarity-matching?WT.mc_id=DSAIroundup-newsletter-davidsmi) with Siamese Networks.
